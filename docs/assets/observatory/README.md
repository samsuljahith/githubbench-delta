# Observatory demo assets

Synthetic longitudinal history for Half-Life Observatory charts and docs.

- `history.jsonl` / `index.json` — 6 cohorts × 2 agents
- Cohort at `2026-07-18` uses **live** mean scores from `exp_6afa2ce533ba4e0a`
  (MiniCPM **0.539**, Codex **0.682**); other cohorts are **synthetic** (labeled in snapshot metadata)
- `sample_report/` — example `benchmark_decay.json`, `benchmark_trends.json`,
  `benchmark_half_life.md`, and Plotly HTML charts

Do not treat synthetic cohorts as real benchmark rankings.
