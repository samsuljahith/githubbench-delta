# Changelog

All notable changes to this project are documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),

## [Unreleased]

### Added

- Phase 8 production hardening: CI workflows, pre-commit, MyPy/coverage gates, Docker prod/dev images, examples, expanded docs, security/community templates, regression and performance smokes.

## [0.1.0] — 2026-07-18

### Added

- Phase 1: scaffolding, config, registries, CLI/API stubs
- Phase 2: agent lifecycle, providers, tools, trajectory, event store
- Phase 3 / 3.5: task framework, 60-task curated corpus, fixtures, strict validation
- Phase 4: 18 deterministic methodology metrics and evaluation engine
- Phase 5: evaluation pipeline, experiment CLI, ResultStore (JSONL/SQLite)
- Phase 6: FastAPI + Plotly dashboard over experiment artifacts
- Phase 7: publication reporting (Markdown/HTML/PDF/JSON/CSV) and compare/regression reports
